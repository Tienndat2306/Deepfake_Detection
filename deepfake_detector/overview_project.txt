  Tóm lại, entrypoint quan trọng nhất là:

  - Preprocess: preprocess/preprocess.py
  - Train: training/train.py
  - Evaluate: evaluation/evaluate.py
  - Model chính: models/deepfake_model.py
  - Dataset chính: data/dataset.py

Luồng chính của dự án này là pipeline phát hiện deepfake theo video, đi từ video gốc -> khuôn mặt đã crop -> model sequence-frame -> checkpoint
  -> đánh giá -> web inference.

  1. Chuẩn bị dữ liệu

  Video gốc được xử lý bởi preprocess/preprocess.py.

  Luồng:

  Video gốc
  -> sample frame đều theo thời gian
  -> detect mặt bằng MediaPipe BlazeFace
  -> align/crop khuôn mặt
  -> resize về kích thước input model
  -> lưu thành ảnh .jpg theo từng video

  Output dataset có dạng:

  dataset/
    train/
      Real/<video_id>/*.jpg
      Fake/<video_id>/*.jpg
    val/
      Real/<video_id>/*.jpg
      Fake/<video_id>/*.jpg
    test/
      Real/<video_id>/*.jpg
      Fake/<video_id>/*.jpg

  Mỗi thư mục video là một sample. Model không học trực tiếp từ file video, mà học từ chuỗi frame khuôn mặt đã được preprocess.

  2. Dataset và augmentation

  File chính: data/dataset.py, data/augmentation.py.

  Khi train:

  DeepfakeDataset
  -> đọc folder video
  -> chọn T frame, ví dụ 10 frame
  -> augmentation ảnh
  -> trả về tensor [T, C, H, W]

  Khi validation/test:

  DeepfakeDataset
  -> tạo nhiều clip từ cùng một video
  -> trả về [N, T, C, H, W]

  Tức là train học theo clip, còn evaluate thường lấy nhiều clip trong một video rồi gộp score lại.

  3. Kiến trúc model

  File chính: models/deepfake_model.py.

  Model là:

  EfficientNet-B4 backbone
  + Transformer temporal head
  + binary classifier

  Forward pass:

  Input: [B, T, C, H, W]

  -> reshape thành [B*T, C, H, W]
  -> EfficientNet trích feature từng frame
  -> reshape lại [B, T, feature_dim]
  -> Transformer học quan hệ thời gian giữa các frame
  -> classifier xuất logit Fake/Real
  -> sigmoid(logit) = xác suất Fake

  Ý tưởng chính: EfficientNet nhìn từng khuôn mặt, Transformer nhìn sự bất thường theo chuỗi thời gian.

  4. Training

  Entrypoint chính: training/train.py.

  Luồng training:

  load configs/train_config.yaml
  + configs/model_config.yaml
  + configs/aug_config.yaml

  -> tạo train/val dataset
  -> tạo DataLoader
  -> tạo DeepfakeDetector
  -> optimizer AdamW
  -> scheduler warmup + cosine
  -> loss FocalLossWithSmoothing
  -> Trainer.fit()

  Trong training/trainer.py, mỗi epoch chạy:

  train_one_epoch()
  -> forward
  -> loss
  -> backward
  -> gradient clipping / AMP nếu có GPU
  -> optimizer step

  validate()
  -> chạy trên validation set
  -> tính AUC/F1/metrics
  -> lưu checkpoint nếu tốt hơn
  -> early stopping nếu không cải thiện

  Checkpoint được lưu trong checkpoints/, ví dụ hiện repo có checkpoint tốt khoảng AUC 0.976999.

  5. Evaluation

  Entrypoint chính: evaluation/evaluate.py.

  Luồng evaluate:

  load checkpoint tốt nhất
  -> load val/test dataset
  -> chạy inference nhiều clip mỗi video
  -> optional TTA
  -> tune threshold trên validation
  -> đánh giá test
  -> xuất metrics/plot/report

  Metrics gồm:

  AUC
  EER
  Average Precision
  Accuracy
  Precision
  Recall
  F1
  Confusion matrix
  ROC curve
  Failure analysis

  Đây là bước kiểm tra model trước khi đem đi demo hoặc deploy.

  6. Inference thực tế / web app

  Phần triển khai hiện tại nằm ở Flask app:

  app.py
  -> app/__init__.py
  -> app/routes.py
  -> app/inference.py

  Luồng khi người dùng upload video:

  POST /api/analyze
  -> lưu video vào app/static/uploads/
  -> DeepfakeInferenceService.analyze_video()
  -> sample frame từ video
  -> detect/crop mặt bằng FaceDetector
  -> lấy đủ num_frames, ví dụ 10 frame
  -> transform giống validation
  -> load checkpoint
  -> model predict
  -> sigmoid ra fake_probability
  -> so threshold 0.5
  -> trả JSON kết quả

  Output thực tế gồm:

  verdict: Fake hoặc Real
  fake_probability
  real_probability
  confidence
  keyframes đã crop
  metadata video
  checkpoint đang dùng
  thời gian xử lý

  Web UI dùng các route như:

  /
   /dashboard
   /api/health
   /api/analyze
   /api/session/<session_id>

  Tóm tắt toàn bộ luồng

  1. Video dataset gốc
     ↓
  2. Preprocess: sample frame -> detect face -> align/crop -> save ảnh mặt
     ↓
  3. Dataset loader: mỗi video = chuỗi frame khuôn mặt
     ↓
  4. Model: EfficientNet trích feature từng frame
     ↓
  5. Transformer học quan hệ thời gian
     ↓
  6. Classifier dự đoán Real/Fake
     ↓
  7. Train + validation -> lưu checkpoint tốt nhất
     ↓
  8. Evaluate trên test set -> metrics/report
     ↓
  9. Flask app: upload video -> preprocess in-memory -> predict -> trả kết quả JSON/UI
